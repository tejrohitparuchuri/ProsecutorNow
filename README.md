## ✨ Beautiful, Dynamic, and Secure

Our platform utilizes a dark, elegant aesthetic with dynamic interactions reminiscent of modern web apps (`#121212` to `#1e1e1e` gradient themes) with smooth, glassmorphism design elements.

<details open>
<summary><b>🔥 Core Functionalities</b></summary>
<br/>

- 📊 **Public Phase Tracker**: Enter a Progress ID to track the phase of any case transparently, while descriptions and personal data remain fiercely protected.
- 🔒 **Secure Case Descriptions**: Authorized parties can view detailed, sensitive case descriptions securely.
- 📂 **My Filed Cases**: A complete log of cases filed under your account, easily accessible in your dashboard.
- ⚖️ **Recruit Prosecutor**: Direct integration for users to securely recruit and assign prosecutors.
- 👨‍⚖️ **Admin Dashboard**: Powerful oversight tools for managing the entire judicial pipeline.

## 🚀 Getting Started

To run the Indian Judicial Case Tracker locally, follow these simple steps:

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/tejrohitparuchuri/ProsecutorNow.git
cd ProsecutorNow
```

### 2️⃣ Install Dependencies
```bash
npm install
```

### 3️⃣ Setup Environment Variables
Create a `.env` file in the root directory and configure your MongoDB connection:
```env
MONGO_URI=your_mongodb_connection_string
PORT=3000
```

### 4️⃣ Start the Application
```bash
# Start the server
npm start
```
The application will launch on `http://localhost:3000`.

## 🎨 Design Philosophy ✨

- **Deep Dark Theme**: Immersive UI with `var(--bg-color-1)` and `var(--bg-color-2)` to reduce eye strain, prioritizing content clarity.
- **Glassmorphism**: Subtle translucency on containers sets a premium, high-tech tone.
- **Micro-Animations**: Animated buttons, pulsing background effects, and sliding modals to create an engaging experience.
